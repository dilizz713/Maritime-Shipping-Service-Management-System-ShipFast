
## Project Purpose - SHIP FAST

The **Ship-Fast** system is designed to streamline and manage maritime shipping operations efficiently. It helps shipping companies track service requests, manage inquiries, generate invoices, handle currency conversions, store documentation securely, conduct online meetings, and generate reportsâ€”all in one integrated platform. The goal is to improve operational efficiency, reduce manual errors, and provide real-time updates for both staff and clients.

## Set up Instructions

**BackEnd**

- Clone the Repository.
- Configure Environment variables - application.properties
- Build and run the backend (backend is running on http://localhost:8080 for API requests)

**FrontEnd**

- Navigate to front end folder
- Open index.html direct on your browser
